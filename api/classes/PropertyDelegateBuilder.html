Class PropertyDelegateBuilder

The builder delegates the accesses of a property to an embedded element.

example

With the builder API - Delegate the accesses of the property value to an embedded input element

import {ElementBuilder, PropertyDelegateBuilder} from "ceb"
class HelloWorld extends HTMLElement {
    value: string = "World"
    connectedCallback() {
        this.innerHTML = `Hello, <input>!`
    }
}
ElementBuilder.get().builder(
    PropertyDelegateBuilder.get("value").to("input")
).register()
example

With the decorator API - Delegate the accesses of the property value to an embedded input element

import {ElementBuilder, PropertyDelegateBuilder} from "ceb"
@ElementBuilder.element()
@PropertyDelegateBuilder.delegate('value', 'input')
class HelloWorld extends HTMLElement {
    value: string = "World"
    connectedCallback() {
        this.innerHTML = `Hello, <input>!`
    }
}
