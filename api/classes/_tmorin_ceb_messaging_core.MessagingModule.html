Class MessagingModule

The module register the component MessagingComponent in order to discover and bootstrap handlers and listeners.

The module expects to resolve a Bus bound with the key BusSymbol.

example

Register the MessagingModule

import {inversion, messaging} from "@tmorin/ceb"
const container = inversion.ContainerBuilder.get()
  .module(new messaging.MessagingModule())
  .build()
